# Italy Trip 2026 Quick Guide / 快速使用指南

Website / 网站: https://italy-trip-2026-cyan.vercel.app

## 中文

这个网站给 4 位同行成员一起查看和更新 Italy Trip 2026 信息。知道链接的人都可以编辑，所以不要录入护照号、证件照、银行卡、完整确认号、二维码、私人电话或不受保护的文件链接。

怎么看信息：

- Dashboard: 看总览、下一项行程、待确认 booking、预算摘要、SOS。
- Itinerary: 看和编辑每天行程。
- Bookings: 看和编辑酒店、交通、门票、保险等预订。
- Budget: 记录已经真实付款的共同花费。
- More: 手机端从这里进入 Packing、Documents、Settings。
- Packing: 管理打包清单。
- Documents: 放私有云文件夹链接和文件准备状态，不上传真实文件。
- Settings: 修改行程名称、日期、成员、路线城市、默认币种。

怎么操作：

- 新增: 点 Add，填写内容，Save。
- 修改: 点对应卡片的 Edit，改完 Save。
- 删除: 点 Delete，并在弹窗确认。
- 文件链接: 如果需要保护，在 Documents 设置 access code；打开时输入 code 后点 Unlock folder。
- 分账: 只有实际付款后才加到 Budget；Itinerary / Bookings 里的 linked expense 会进入 Budget 汇总。

## English

This website is for the 4 travelers to view and update Italy Trip 2026 information together. Anyone with the link can edit it, so do not enter passport numbers, ID scans, card details, full confirmation numbers, QR codes, private phone numbers, or unprotected private file links.

Where to look:

- Dashboard: overview, next itinerary item, booking alerts, budget summary, SOS.
- Itinerary: view and edit daily plans.
- Bookings: view and edit hotels, transport, tickets, insurance, and reservations.
- Budget: record real shared expenses after payment.
- More: on phones, open Packing, Documents, and Settings from here.
- Packing: manage the packing checklist.
- Documents: store private cloud folder links and document status; do not upload real files here.
- Settings: update trip name, dates, travelers, route cities, and default currencies.

How to use:

- Add: tap Add, fill in the form, Save.
- Edit: tap Edit on the card, update it, Save.
- Delete: tap Delete and confirm.
- Folder links: in Documents, use an access code if needed; enter the code and tap Unlock folder.
- Expenses: add to Budget only after payment. Linked expenses from Itinerary / Bookings are included in Budget.
